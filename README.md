🔧 Instalação

    Clone o repositório:

bash

git clone https://github.com/seu-usuario/nome-do-repositorio.git

    Instale as dependências:

bash

npm install
# ou
yarn install

    Configure o ambiente:

bash

cp .env.example .env
# Edite o .env com suas configurações

    Execute em desenvolvimento:

bash

npm run dev
# ou
yarn dev
